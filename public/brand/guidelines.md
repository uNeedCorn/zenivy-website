# Zenivy Brand Guidelines

> 品牌識別使用規範

---

## 品牌名稱

### 名稱

**Zenivy** `/ˈzɛnɪvi/`

### 命名由來

| 組成 | 意義 |
|------|------|
| **Zen** | 禪、專注、平衡 |
| **ivy** | 常春藤、關係生長、連結 |

> Zenivy 象徵以禪意的專注態度，培養如常春藤般持續生長的人際關係。

### 使用規範

- ✅ 首字母大寫：**Zenivy**
- ❌ 全大寫：~~ZENIVY~~
- ❌ 全小寫：~~zenivy~~
- ❌ 分開寫：~~Zen ivy~~

## Logo

### 設計概念

Logo 由兩個元素組成：

1. **五邊形** - 傾斜 6° 的不規則五邊形，象徵動態與成長
2. **樹枝圖案** - 簡化的樹枝/常春藤，代表人際網絡的分支與延伸

### Logo 版本

Logo 有兩個版本，分別用於淺色和深色背景：

#### 填充版（Filled）- 淺色模式

```
┌────────────────┐
│   ◆◆◆◆◆◆◆◆    │  五邊形：Teal 500 填充
│   ◆ ╲  ╱ ◆    │  樹枝：白色線條
│   ◆  ╲╱  ◆    │
│   ◆  │   ◆    │
│   ◆ ╱ ╲  ◆    │
│   ◆◆◆◆◆◆◆◆    │
└────────────────┘
```

- **五邊形**：`#14B8A6` (Teal 500) 填充
- **樹枝線條**：白色 (`#FFFFFF`)
- **使用情境**：淺色背景、App Icon

#### 描邊版（Outline）- 深色模式

```
┌────────────────┐
│   ◇◇◇◇◇◇◇◇    │  五邊形：Teal 400 描邊（無填充）
│   ◇ ╲  ╱ ◇    │  樹枝：Teal 400 線條
│   ◇  ╲╱  ◇    │
│   ◇  │   ◇    │
│   ◇ ╱ ╲  ◇    │
│   ◇◇◇◇◇◇◇◇    │
└────────────────┘
```

- **五邊形**：`#2DD4BF` (Teal 400) 描邊，無填充
- **樹枝線條**：`#2DD4BF` (Teal 400)
- **使用情境**：深色背景、深色模式 UI

### 檔案位置

| 檔案 | 路徑 | 用途 |
|------|------|------|
| Logo 填充版 | `doc/brand/logo/logo.svg` | 淺色模式、App Icon |
| Logo 描邊版 | `doc/brand/logo/logo-outline.svg` | 深色模式 |
| Icon 1024px | `doc/brand/icon/icon-1024.png` | App Icon 原始檔 |

### 使用規範

#### 版本選擇

| 背景 | 建議版本 | 檔案 |
|------|----------|------|
| 淺色背景 | 填充版 | `logo.svg` |
| 深色背景 | 描邊版 | `logo-outline.svg` |
| App Icon | 填充版 | `logo.svg` |

#### 最小尺寸

- 數位：最小 24px
- 印刷：最小 10mm

#### 安全區域

Logo 周圍應保持至少 Logo 高度 25% 的淨空區域。

#### 禁止事項

- ❌ 在淺色背景使用描邊版（對比度不足）
- ❌ 在深色背景使用填充版（視覺過重）
- ❌ 修改 Logo 顏色
- ❌ 添加陰影或特效
- ❌ 拉伸或扭曲比例

## App Icon

### 規格

- **形狀**：透明背景，讓系統套用圓角
- **尺寸**：1024x1024px（原始檔）
- **格式**：PNG（透明）、ICNS（macOS）、ICO（Windows）

### 生成流程

使用腳本從原始 SVG 生成各平台所需格式：

```bash
node scripts/generate-icon.mjs
```

輸出位置：`build/icon.png`、`build/icon.icns`、`build/icon.ico`
